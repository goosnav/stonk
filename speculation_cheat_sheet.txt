STOCK TRADER SPECULATION CHEAT SHEET
Goal: identify statistically repeatable edge net of slippage, spread, taxes, borrow, and commissions.
Core rule: no indicator predicts by itself; edge = signal + regime + sizing + execution + risk control.

1) CORE RETURN, RISK, AND EDGE EQUATIONS
================================================================================

Simple Return: r_t = P_t/P_{t-1} - 1; basic percentage return from one period to next.

Log Return: l_t = ln(P_t/P_{t-1}); additive return used in quantitative modeling.

Cumulative Return: CR = product(1+r_t) - 1; total compounded return across periods.

Annualized Return: AR = (1+CR)^(PeriodsPerYear/N) - 1; scales return to yearly basis.

Annualized Volatility: sigma_ann = std(r_t) * sqrt(PeriodsPerYear); standard annual risk measure.

Expected Value: EV = p_win*avg_win - p_loss*avg_loss; expected profit per trade before/after costs.

Profit Factor: PF = gross_profit / gross_loss; values above 1 mean gross wins exceed gross losses.

Win Rate: WR = winning_trades / total_trades; useful only with payoff ratio.

Payoff Ratio: PR = avg_win / avg_loss; determines whether win rate is sufficient.

Breakeven Win Rate: BE_WR = avg_loss / (avg_win + avg_loss); minimum win rate needed before costs.

Sharpe Ratio: Sharpe = mean(r_t - R_f) / std(r_t - R_f); return per unit total volatility.

Sortino Ratio: Sortino = mean(r_t - R_f) / downside_deviation; return per unit downside volatility.

Information Ratio: IR = mean(r_p - r_b) / tracking_error; active return per unit benchmark-relative risk.

Treynor Ratio: Treynor = mean(r_p - R_f) / beta_p; return per unit market beta.

Calmar Ratio: Calmar = annual_return / max_drawdown; return relative to worst peak-to-trough loss.

Omega Ratio: Omega = gains_above_threshold / losses_below_threshold; distribution-aware risk/reward.

Skewness: skew = E[(r-mu)^3]/sigma^3; measures asymmetric return tails.

Kurtosis: kurt = E[(r-mu)^4]/sigma^4; measures fat-tailed return behavior.

Maximum Drawdown: MDD = min((Equity_t - Peak_t)/Peak_t); largest historical equity decline.

Value at Risk: VaR_alpha = percentile_loss at confidence alpha; estimates threshold loss.

Conditional VaR: CVaR = average loss beyond VaR; expected tail loss.

Beta: beta_i = cov(r_i,r_m)/var(r_m); sensitivity to market movement.

Alpha: alpha_i = r_i - [R_f + beta_i*(R_m-R_f)]; benchmark-adjusted excess return.

Jensen Alpha: alpha = R_p - [R_f + beta_p*(R_m-R_f)]; portfolio excess return over Capital Asset Pricing Model.

Tracking Error: TE = std(r_p - r_b); volatility of active returns versus benchmark.

Turnover: TO = sum(abs(w_t - w_{t-1})); trading intensity that drives costs/taxes.

Capacity: Capacity ~= alpha_decay / market_impact; rough maximum capital before edge degrades.

Slippage: Slippage = execution_price - decision_price; realized execution drag.

Implementation Shortfall: IS = paper_return - actual_return; total cost of execution delay/slippage/fees.

Kelly Fraction: f* = (bp - q)/b; b=win/loss payoff ratio, p=win probability, q=1-p.

Fractional Kelly: f = c*f*, 0<c<1; reduced Kelly sizing to lower drawdown risk.

Optimal f: position fraction maximizing geometric growth; aggressive and usually unstable.

Geometric Mean Return: G = product(1+r_t)^(1/N)-1; compounded average return.

Arithmetic Mean Return: A = mean(r_t); simple average return, often overstates long-run compounding.

Risk of Ruin: RoR approximates probability of capital falling below survival threshold; used for sizing.

Expected Log Growth: G = E[ln(1+f*r)]; core Kelly/objective function.

2) PRICE, MOMENTUM, AND TREND INDICATORS
================================================================================

SMA: SMA_N = mean(C_t...C_{t-N+1}); simple moving average trend filter.

EMA: EMA_t = alpha*C_t + (1-alpha)*EMA_{t-1}, alpha=2/(N+1); faster moving average.

WMA: WMA = sum(weight_i*C_i)/sum(weight_i); gives recent prices more weight.

HMA: Hull Moving Average reduces lag using weighted moving averages; used for smoother trend turns.

KAMA: Kaufman Adaptive Moving Average changes sensitivity based on noise/trend efficiency.

VWMA: VWMA_N = sum(C_t*V_t)/sum(V_t); volume-weighted moving average.

Price Above MA: Signal = C_t > MA_N; basic trend-following regime filter.

Golden Cross: SMA_50 crosses above SMA_200; long-term bullish trend signal.

Death Cross: SMA_50 crosses below SMA_200; long-term bearish trend signal.

MA Slope: Slope = MA_t - MA_{t-N}; trend direction/strength.

Rate of Change: ROC_N = C_t/C_{t-N} - 1; simple momentum over N periods.

Momentum: MOM_N = C_t - C_{t-N}; absolute price change over lookback.

Time-Series Momentum: Signal = sign(C_t/C_{t-N}-1); buy positive own-trend, sell/avoid negative.

Cross-Sectional Momentum: rank stocks by past return; buy winners/short losers.

Relative Strength: RS = stock return / benchmark return; identifies leadership versus market.

RSI: RSI = 100 - 100/(1+RS), RS=avg_gain/avg_loss; overbought/oversold momentum oscillator.

Stochastic %K: %K = 100*(C-L_N)/(H_N-L_N); close location within recent range.

Stochastic %D: %D = SMA(%K); smoothed stochastic signal line.

MACD: MACD = EMA_12 - EMA_26; trend/momentum spread.

MACD Signal: Signal = EMA_9(MACD); crossover trigger.

MACD Histogram: Hist = MACD - Signal; momentum acceleration proxy.

ADX: Average Directional Index measures trend strength independent of direction.

DMI +DI/-DI: Directional Movement Index compares positive and negative directional movement.

Aroon Up: 100*(N-periods_since_N_high)/N; measures recent high proximity.

Aroon Down: 100*(N-periods_since_N_low)/N; measures recent low proximity.

TRIX: triple-smoothed EMA rate of change; filters noisy momentum.

TSI: True Strength Index uses double-smoothed momentum; trend/momentum oscillator.

CCI: CCI = (TypicalPrice - SMA(TP))/(0.015*MeanDeviation); detects deviations from average price.

Williams %R: %R = -100*(H_N-C)/(H_N-L_N); overbought/oversold range oscillator.

Ultimate Oscillator: combines short/intermediate/long buying pressure; reduces timeframe bias.

Chande Momentum Oscillator: CMO = 100*(sum_gains-sum_losses)/(sum_gains+sum_losses); RSI-like oscillator.

Detrended Price Oscillator: DPO = C_t - SMA shifted; identifies cycle deviations.

Know Sure Thing: KST combines multiple rate-of-change periods; long-term momentum indicator.

Ichimoku Tenkan-sen: (9-period high + 9-period low)/2; short trend equilibrium.

Ichimoku Kijun-sen: (26-period high + 26-period low)/2; medium trend equilibrium.

Ichimoku Senkou Span A: (Tenkan + Kijun)/2 projected forward; cloud boundary.

Ichimoku Senkou Span B: (52-period high + 52-period low)/2 projected forward; cloud boundary.

Ichimoku Cloud: price above cloud = bullish regime, below cloud = bearish regime.

Parabolic SAR: trailing stop that accelerates with trend; trend-following exit/flip indicator.

Supertrend: ATR-based trend line; bullish when price closes above band.

Donchian Channel: upper=max(H,N), lower=min(L,N); breakout system channel.

Turtle Breakout: buy N-day high, sell/short N-day low; classic trend-following rule.

Keltner Channel: EMA +/- multiplier*ATR; trend/volatility envelope.

Bollinger Bands: SMA_N +/- k*std(C,N); volatility envelope around price.

Bollinger %B: %B = (C-LowerBand)/(UpperBand-LowerBand); price location inside bands.

Bollinger Bandwidth: BW = (Upper-Lower)/Middle; volatility compression/expansion gauge.

Moving Average Envelope: MA +/- fixed percentage; simple overextension band.

Z-Score Price: z = (C - mean(C,N))/std(C,N); normalized deviation from recent mean.

Linear Regression Slope: slope of C over N bars; trend strength and direction.

R-Squared Trend: R^2 of linear regression; measures trend fit quality.

Price Channel Breakout: enter when C exceeds prior N high/low; captures volatility expansions.

Gap Momentum: gap = O_t/C_{t-1}-1; overnight information shock measure.

52-Week High Signal: C near max(C,252); persistent momentum/attention effect.

52-Week Low Signal: C near min(C,252); distress/reversal or continuation signal.

Relative Moving Average Spread: (SMA_short/SMA_long)-1; normalized trend state.

Distance From VWAP: (P_t - VWAP_t)/VWAP_t; intraday overextension or trend signal.

Distance From High: C_t/max(C,N)-1; pullback depth from recent peak.

Distance From Low: C_t/min(C,N)-1; rebound strength from recent trough.

================================================================================
3) MEAN REVERSION INDICATORS
================================================================================

Z-Score Mean Reversion: z=(P-mean(P,N))/std(P,N); fade extreme deviations if regime is range-bound.

Bollinger Reversion: short/fade upper band or buy lower band; works best in low-trend regimes.

RSI Reversion: RSI<30 buy, RSI>70 sell/avoid; crude oversold/overbought rule.

Stochastic Reversion: %K below 20 or above 80; range-bound turning signal.

Pairs Spread: spread = y - beta*x; relative mispricing between cointegrated assets.

Spread Z-Score: z = (spread - mean(spread,N))/std(spread,N); entry/exit measure for pairs.

Half-Life of Mean Reversion: h = -ln(2)/ln(phi), phi=AR(1) coefficient; expected decay speed.

Ornstein-Uhlenbeck Model: dX = theta*(mu-X)dt + sigma*dW; continuous mean-reversion model.

Residual Reversion: residual = stock_return - factor_model_return; trade extreme idiosyncratic deviations.

Intraday VWAP Reversion: fade large deviations from VWAP; common in liquid large caps.

Open-to-Close Reversion: fade excessive opening gap if no follow-through; intraday stat-arb setup.

Close-to-Open Reversion: buy weak close/sell strong close for overnight reversal effects.

Short-Term Reversal: prior 1-5 day losers rebound and winners cool; common equity microstructure anomaly.

Overreaction Reversal: extreme single-day move partially reverses; requires liquidity/news filters.

Earnings Drift Reversal Filter: avoid fading post-earnings drift too early; event momentum can dominate.

Volume-Climax Reversal: extreme volume plus price exhaustion; possible capitulation/short squeeze turn.

Capitulation Signal: large down move + huge volume + high put/call + high volatility; possible reversal.

Breadth Washout: market-wide oversold breadth; index/sector mean reversion trigger.

================================================================================
4) VOLATILITY, RANGE, AND RISK INDICATORS
================================================================================

True Range: TR = max(H-L, abs(H-C_{t-1}), abs(L-C_{t-1})); captures gap-adjusted range.

ATR: ATR_N = average(TR,N); average absolute price range.

ATR Percent: ATRP = ATR/C; normalized volatility.

Realized Volatility: RV = std(l_t,N)*sqrt(PeriodsPerYear); historical volatility.

Parkinson Volatility: sigma^2 = [1/(4N ln2)]*sum(ln(H/L)^2); high-low volatility estimator.

Garman-Klass Volatility: uses O,H,L,C to estimate volatility more efficiently than close-close.

Rogers-Satchell Volatility: OHLC volatility estimator robust to drift.

Yang-Zhang Volatility: combines overnight and intraday volatility; useful for equities.

Volatility Regime: current RV percentile versus history; determines trend/reversion suitability.

Volatility Breakout: enter when price exceeds range by ATR multiple; captures expansion.

Volatility Compression: low Bollinger Bandwidth/ATR percentile; setup for future expansion.

ATR Stop: stop = entry - k*ATR for longs; volatility-adjusted exit.

Chandelier Exit: stop = highest_high_N - k*ATR; trend-following trailing stop.

Donchian Exit: exit on N-day low/high; classic trend system risk control.

HV-IV Spread: HV - IV; compares realized volatility to option-implied volatility.

IV Rank: IVR = (current_IV - IV_low_1y)/(IV_high_1y - IV_low_1y); option volatility richness.

IV Percentile: percent of days IV was below current IV; volatility regime rank.

VIX: S&P 500 implied volatility index; broad equity fear/risk gauge.

VVIX: volatility of VIX options; meta-volatility/fear convexity gauge.

MOVE Index: Treasury implied volatility; rates volatility stress indicator.

Vol-of-Vol: std(volatility changes); instability gauge.

GARCH(1,1): sigma_t^2 = omega + alpha*epsilon_{t-1}^2 + beta*sigma_{t-1}^2; volatility clustering model.

EGARCH: log variance model capturing leverage/asymmetry effects.

GJR-GARCH: GARCH with asymmetric response to negative shocks.

Stochastic Volatility Model: volatility follows latent random process; used in options/quant models.

Realized Beta: rolling beta estimated from intraday/daily returns; dynamic market exposure.

Realized Correlation: rolling correlation among assets; increases in crises.

Average Correlation: mean pairwise stock correlation; high values indicate macro/risk-off regime.

5) VOLUME, FLOW, AND LIQUIDITY INDICATORS
================================================================================

Volume SMA: average(V,N); baseline activity.

Relative Volume: RVOL = V_t / average(V,N); unusual trading activity indicator.

Dollar Volume: DollarVol = P_t*V_t; liquidity/capacity measure.

ADV: average daily volume; tradability/capacity filter.

Volume Spike: V_t > k*avg(V,N); attention/information shock marker.

On-Balance Volume: OBV_t = OBV_{t-1} +/- V_t; cumulative volume pressure.

Volume Price Trend: VPT_t = VPT_{t-1}+V_t*(C_t-C_{t-1})/C_{t-1}; volume-weighted price move.

Accumulation/Distribution Line: ADL uses close location in range times volume; buying/selling pressure.

Chaikin Money Flow: CMF = sum(MoneyFlowVolume,N)/sum(V,N); volume-weighted accumulation.

Money Flow Index: MFI = volume-weighted RSI using typical price; overbought/oversold with volume.

Ease of Movement: relates price change to volume; detects low-resistance price movement.

Force Index: FI = V_t*(C_t-C_{t-1}); volume-weighted momentum.

Negative Volume Index: NVI tracks price change on lower-volume days; smart-money folklore indicator.

Positive Volume Index: PVI tracks price change on higher-volume days; crowd participation indicator.

VWAP: VWAP = sum(P_t*V_t)/sum(V_t); institutional benchmark and intraday anchor.

Anchored VWAP: VWAP from a selected event/date; identifies post-event cost basis.

Volume Profile: volume traded at each price level; detects high-volume nodes/support/resistance.

Market Profile: time spent at each price level; auction-market structure tool.

POC: Point of Control = price with highest volume/time; key acceptance price.

Value Area: price range containing ~70% of volume/time; auction equilibrium zone.

Cumulative Volume Delta: CVD = cumulative aggressive buy volume - aggressive sell volume; order-flow pressure.

Delta Divergence: price makes new high while CVD fails; possible exhaustion.

Block Trade Detection: unusually large prints; institutional activity clue.

Dark Pool Prints: off-exchange block activity; possible accumulation/distribution signal.

ETF Flow: net creations/redemptions; fund-level demand/supply pressure.

Fund Flow: net money into mutual funds/ETFs; medium-term demand indicator.

6) BREADTH AND MARKET INTERNALS
================================================================================

Advance-Decline Line: cumulative advancing stocks - declining stocks; market participation gauge.

Advance/Decline Ratio: advancing issues / declining issues; daily breadth strength.

New Highs-New Lows: count of 52-week highs minus lows; market health indicator.

McClellan Oscillator: EMA_19(adv-dec) - EMA_39(adv-dec); breadth momentum.

McClellan Summation Index: cumulative McClellan Oscillator; intermediate breadth trend.

Percent Above 50DMA: stocks above 50-day moving average; breadth regime indicator.

Percent Above 200DMA: stocks above 200-day moving average; long-term market health.

Arms Index/TRIN: (Adv/Decl)/(AdvVol/DeclVol); panic/overbought breadth-volume indicator.

Up/Down Volume Ratio: up-volume / down-volume; conviction of market move.

Breadth Thrust: sudden surge in advancing stocks; rare strong bullish regime signal.

Hindenburg Omen: simultaneous new highs/lows under specific conditions; crash-risk folklore indicator.

Sector Breadth: percent of sector stocks in uptrends; sector rotation confirmation.

Equal-Weight vs Cap-Weight Ratio: RSP/SPY or equal-weight index / cap-weight index; market concentration gauge.

Small-Cap/Large-Cap Ratio: IWM/SPY; risk appetite/market breadth proxy.

Cyclical/Defensive Ratio: cyclicals divided by defensives; economic risk-on/off gauge.

High Beta/Low Vol Ratio: SPHB/SPLV; speculative appetite proxy.

Semiconductor/Market Ratio: SMH/SPY; tech-cycle leadership indicator.

Credit/Equity Confirmation: HYG/SPY or LQD/SPY; credit confirming/risk warning.

7) SUPPORT, RESISTANCE, STRUCTURE, AND CHART TOOLS
================================================================================

Horizontal Support: prior price zone where demand appeared; stop/entry reference.

Horizontal Resistance: prior price zone where supply appeared; breakout/fade reference.

Trendline: line connecting swing highs/lows; discretionary trend structure.

Pivot High: local high above neighboring highs; swing structure point.

Pivot Low: local low below neighboring lows; swing structure point.

Higher Highs/Higher Lows: bullish market structure; trend confirmation.

Lower Highs/Lower Lows: bearish market structure; downtrend confirmation.

Break of Structure: price violates prior swing high/low; trend change signal.

Market Structure Shift: failure of prior trend sequence; reversal clue.

Liquidity Sweep: price briefly breaks prior high/low then reverses; stop-run pattern.

False Breakout: breakout fails and returns into range; reversal setup.

Breakout Retest: price breaks level, returns to test it, then continues; trend-entry setup.

Measured Move: target = breakout point + prior range height; chart projection.

Fibonacci Retracement: levels 23.6/38.2/50/61.8/78.6%; common discretionary pullback zones.

Fibonacci Extension: 127.2/161.8/261.8%; discretionary target zones.

Pivot Points: PP=(H+L+C)/3; intraday support/resistance framework.

Classic Pivot R1: R1=2*PP-L; first resistance.

Classic Pivot S1: S1=2*PP-H; first support.

Classic Pivot R2: R2=PP+(H-L); second resistance.

Classic Pivot S2: S2=PP-(H-L); second support.

Opening Range Breakout: trade break above/below first X-minute range; intraday momentum setup.

Initial Balance: first hour range; futures/market-profile structure.

Gap Fill: price returns to prior close after opening gap; common intraday target concept.

Unfilled Gap: gap remains open; possible momentum/imbalance signal.

Supply Zone: prior sharp decline area; discretionary resistance.

Demand Zone: prior sharp rally area; discretionary support.

Wyckoff Accumulation: range structure suggesting institutional accumulation.

Wyckoff Distribution: range structure suggesting institutional selling.

Spring: false breakdown in Wyckoff range; reversal trigger.

Upthrust: false breakout in Wyckoff range; bearish reversal trigger.

8) CANDLESTICK AND PRICE ACTION PATTERNS
================================================================================

Doji: open and close nearly equal; indecision, useful only at key levels.

Hammer: small body with long lower wick; possible rejection of lows.

Shooting Star: small body with long upper wick; possible rejection of highs.

Engulfing Bullish: green candle body engulfs prior red body; short-term reversal signal.

Engulfing Bearish: red candle body engulfs prior green body; short-term reversal signal.

Morning Star: three-candle bullish reversal pattern; better after downtrend.

Evening Star: three-candle bearish reversal pattern; better after uptrend.

Harami: small candle inside prior body; volatility contraction/indecision.

Inside Bar: full bar range inside prior bar; compression setup.

Outside Bar: bar exceeds prior high and low; volatility expansion signal.

Marubozu: large body with tiny wicks; strong directional conviction.

Pin Bar: long wick rejecting level; discretionary reversal signal.

Three White Soldiers: three strong up candles; momentum continuation.

Three Black Crows: three strong down candles; bearish continuation/reversal.

Exhaustion Candle: huge candle after extended move; possible terminal move.

Narrow Range 7: current range narrowest of last 7 bars; volatility compression.

Wide Range Bar: range much larger than average; possible information shock.

9) STATISTICAL ARBITRAGE AND RELATIVE-VALUE METHODS
================================================================================

Pairs Trading: long undervalued asset and short overvalued paired asset; relies on spread mean reversion.

Cointegration: stationary linear combination of non-stationary prices; validates long-run pair relation.

Engle-Granger Test: regression residual stationarity test for cointegration.

Johansen Test: multivariate cointegration test for baskets.

Spread Regression: y_t = alpha + beta*x_t + epsilon_t; beta hedge ratio estimated by regression.

Hedge Ratio: beta = cov(y,x)/var(x); amount of x used to hedge y.

ADF Test: tests whether time series has unit root; used for stationarity.

KPSS Test: tests stationarity directly; complements ADF.

Hurst Exponent: H<0.5 mean-reverting, H=0.5 random walk, H>0.5 trending.

Variance Ratio Test: tests random walk vs mean reversion/momentum.

Z-Score Entry: enter when spread z exceeds +/-threshold; standard stat-arb trigger.

Z-Score Exit: exit when spread z returns near 0; mean-reversion profit capture.

Basket Trading: trade one asset against weighted basket; broader relative-value structure.

PCA Residual Trading: use principal components to model common factors; trade residual dislocations.

Cluster Pairs: pair stocks with similar factor/industry behavior; improves economic validity.

Sector Neutral Long/Short: long top-ranked names and short bottom-ranked names within sector.

Market Neutral Long/Short: beta-adjusted book with near-zero market exposure.

Dollar Neutral Portfolio: long dollars equal short dollars; controls gross market direction less precisely than beta neutrality.

Beta Neutral Portfolio: sum(w_i*beta_i)=0; removes market beta exposure.

Factor Neutral Portfolio: constrain exposures to value/momentum/size/etc.; isolate target alpha.

Residual Momentum: momentum of stock after removing factor/industry returns; cleaner alpha signal.

Residual Reversal: mean reversion in idiosyncratic residual returns; common stat-arb signal.

Lead-Lag Trading: use faster-moving related asset to predict slower asset; common in ETFs/futures/stocks.

ETF Constituents Arb: trade ETF versus underlying basket dislocations; institutional/HFT domain.

Dual-Listed Arb: trade pricing differences across listings; mostly HFT/institutional.

Index Arb: futures/ETF/index basket mispricing; execution-speed dependent.

ADR Arb: American Depositary Receipt versus local shares; currency/liquidity-adjusted relative value.

10) FACTOR MODELS AND EQUITY FACTORS
================================================================================

CAPM: R_i-R_f = alpha + beta*(R_m-R_f) + epsilon; market-beta return model.

Fama-French 3-Factor: R_i-R_f = alpha + beta_m*MKT + beta_s*SMB + beta_v*HML + epsilon; market/size/value model.

Carhart 4-Factor: adds MOM momentum factor to Fama-French; common performance attribution model.

Fama-French 5-Factor: adds profitability RMW and investment CMA; broader equity factor model.

q-Factor Model: market, size, investment, profitability; alternative factor framework.

AQR Style Factors: value, momentum, quality, defensive/low-risk; practical quant framework.

Barra Risk Model: commercial multi-factor model for portfolio risk/exposure.

Size Factor: small caps historically earn premium but are cyclical/liquidity-sensitive.

Value Factor: cheap stocks outperform expensive stocks over long horizons but suffer long drawdowns.

Momentum Factor: recent winners continue outperforming; one of the strongest empirical factors.

Quality Factor: profitable, stable, low-leverage firms outperform junk over cycles.

Low Volatility Factor: lower-vol stocks can outperform risk-adjusted; crowding-sensitive.

Profitability Factor: high operating profitability predicts stronger returns.

Investment Factor: low asset growth/conservative investment tends to outperform.

Accruals Factor: low accruals outperform high accruals; earnings quality anomaly.

Net Issuance Factor: firms issuing shares underperform repurchasers.

Buyback Yield: repurchases / market cap; shareholder return signal.

Shareholder Yield: dividends + buybacks - issuance; broader capital return signal.

Earnings Yield: EPS / Price; inverse P/E valuation signal.

Book-to-Market: BV / MCap; classic value signal.

Sales-to-Price: Revenue / MCap; value metric less sensitive to accounting profits.

Cash-Flow-to-Price: Operating Cash Flow / MCap; value/quality hybrid.

FCF Yield: FCF / MCap or FCF / EV; free cash flow valuation signal.

EV/EBITDA: enterprise value divided by EBITDA; capital-structure-neutral valuation multiple.

EV/Sales: enterprise value divided by revenue; common for unprofitable growth firms.

PEG Ratio: P/E divided by EPS growth; growth-adjusted valuation metric.

Gross Profitability: gross profit / total assets; strong quality factor.

ROE: NI / equity; profitability metric.

ROA: NI / assets; asset efficiency/profitability metric.

ROIC: NOPAT / invested capital; capital efficiency metric.

Debt/Equity: total debt / equity; leverage risk metric.

Net Debt/EBITDA: net debt / EBITDA; solvency/leverage metric.

Interest Coverage: EBIT / interest expense; ability to service debt.

Piotroski F-Score: 0-9 accounting quality/value score; screens improving cheap companies.

Altman Z-Score: bankruptcy risk formula; distress filter.

Beneish M-Score: earnings manipulation risk model; accounting red-flag filter.

Sloan Accruals: accruals / assets; high accruals imply lower earnings quality.

Earnings Revision Factor: upward analyst EPS revisions; strong medium-term predictor.

Estimate Dispersion: std(analyst estimates)/mean estimate; uncertainty/disagreement signal.

Short Interest Factor: shares short / float; bearish sentiment and squeeze-risk input.

Analyst Upgrade/Downgrade: sell-side rating change; event/flow signal.

Target Price Revision: change in analyst price target; sentiment/revision signal.

Institutional Ownership Change: change in institutional holdings; slow flow signal.

Insider Buying: executives/directors buying shares; generally stronger than insider selling.

11) FUNDAMENTAL, QUALITY, AND VALUATION TOOLS
================================================================================

Discounted Cash Flow: Value = sum(FCF_t/(1+r)^t) + TerminalValue/(1+r)^T; intrinsic value model.

Dividend Discount Model: Value = D_1/(r-g); values dividend stream with growth g.

Residual Income Model: Value = BV + PV(future residual income); accounting-based valuation.

Breadth Regime: breadth thrust/washout/internal deterioration; participation state.

Beta Targeting: adjust exposure to desired beta; controls market sensitivity.

Gross Exposure: sum(abs(long)+abs(short)); total market deployed.

Net Exposure: long exposure - short exposure; directional market exposure.

Leverage Constraint: max gross exposure; protects against liquidation.

Stop Loss: exit at predefined adverse price; loss containment.

Volatility Stop: stop distance based on ATR/RV; adapts to noise.

Time Stop: exit after N periods if thesis fails; reduces capital stagnation.

Trailing Stop: stop follows favorable price movement; locks trend gains.

Profit Target: predefined favorable exit level; useful in mean-reversion systems.

Scale-In: add to position as thesis confirms or price improves; increases complexity/risk.

Scale-Out: reduce position as target approaches; lowers reversal risk.

Pyramiding: add to winners; trend-following tactic.

Averaging Down: add to losers; dangerous unless statistically validated.

Rebalancing: reset weights periodically; harvests drift and controls risk.

Risk Budget: maximum portfolio/trade risk allocation.

Drawdown Kill Switch: stop trading after loss threshold; protects from regime/model failure.

Daily Loss Limit: max loss per day; prevents emotional/operational blowups.

Exposure Cap by Sector: prevents concentration in correlated names.

Correlation Cap: limits positions that move together.

Liquidity Cap: position no more than x% of ADV; reduces market impact.

25) EXECUTION ALGORITHMS AND ORDER TOOLS
================================================================================

Market Order: immediate execution; high certainty, high slippage risk.

Limit Order: max buy/min sell price; price control, fill uncertainty.

Stop Order: becomes market order after trigger; risk exit/momentum entry.

Stop-Limit Order: stop trigger creates limit order; avoids bad fills but may not execute.

Market-on-Open: execute at opening auction; useful for open-price signals.

Market-on-Close: execute at closing auction; useful for close-price signals.

VWAP Algo: targets volume-weighted average price; standard institutional benchmark.

POV Algo: participation-of-volume; trades fixed percentage of market volume.

Implementation Shortfall Algo: minimizes slippage versus decision price.

Iceberg Order: shows small quantity while hiding full size; reduces information leakage.

Pegged Order: price pegged to bid/ask/mid; adaptive passive execution.

Midpoint Peg: hidden midpoint execution; saves spread if filled.

Dark Pool Routing: seeks hidden liquidity off-exchange; reduces displayed impact.

Smart Routing: chooses venue/order type dynamically; execution optimization.

Liquidity-Seeking Algo: aggressively searches for available liquidity.

Arrival Price Benchmark: decision-time price benchmark; used for execution quality.

Close Price Benchmark: closing price benchmark; common for index/portfolio trades.

Participation Rate: order size / market volume; controls impact.

Market Impact Model: estimated price movement caused by own trade.

Almgren-Chriss Model: optimal execution balancing impact and risk.

Square-Root Impact Law: impact roughly proportional to sigma*sqrt(Q/ADV); empirical execution model.

29) SPECIFIC EQUATIONS USED IN QUANT/TRADING RESEARCH
================================================================================

OLS Regression: y = Xb + e; estimates linear relation between features X and target y.

OLS Coefficients: b = (X'X)^(-1)X'y; closed-form least-squares estimate.

Correlation: rho_xy = cov(x,y)/(sigma_x*sigma_y); linear co-movement.

Covariance Matrix: Sigma_ij = cov(r_i,r_j); required for portfolio risk.

Portfolio Return: R_p = sum(w_i*r_i); weighted asset return.

Portfolio Variance: sigma_p^2 = w' Sigma w; portfolio risk from weights/covariances.

Marginal Risk Contribution: MRC_i = (Sigma w)_i / sigma_p; asset contribution to portfolio risk.

Risk Contribution: RC_i = w_i*(Sigma w)_i / sigma_p; total risk share by asset.

Mean-Variance Objective: maximize w'mu - lambda*w'Sigma*w; return-risk portfolio optimization.

Minimum Variance Portfolio: minimize w'Sigma*w subject to constraints; lowest-vol allocation.

Tangency Portfolio: w proportional to Sigma^(-1)*(mu-R_f); max-Sharpe unconstrained portfolio.

Black-Litterman: combines market-implied returns with investor views; stabilized allocation model.

CAPM Expected Return: E[R_i]=R_f+beta_i*(E[R_m]-R_f); benchmark expected-return model.

Logistic Function: p=1/(1+e^(-z)); maps score to probability.

Z-Score: z=(x-mu)/sigma; normalized deviation.

t-Stat: t=mean(x)/(std(x)/sqrt(N)); mean significance estimate.

Information Coefficient: IC = corr(predicted_return, realized_return); forecast quality measure.

ICIR: mean(IC)/std(IC); stability of predictive rank signal.

Rank IC: Spearman correlation between predicted and realized ranks; common quant metric.

Hit Rate: fraction(sign(prediction)=sign(return)); directional accuracy.

Mean Absolute Error: MAE=mean(abs(y-yhat)); forecast error metric.

Root Mean Squared Error: RMSE=sqrt(mean((y-yhat)^2)); penalizes large errors.

Cross Entropy: -mean[y ln(p)+(1-y)ln(1-p)]; classification loss.

Hedge Ratio: h = cov(S,F)/var(F); futures/ETF hedge ratio.

Beta Hedge Shares: shares_hedge = beta*position_value/hedge_price; market hedge sizing.

Delta Hedge Shares: shares = -option_delta*contracts*100; neutralizes option delta.

Expected Shortfall: ES_alpha=E[Loss | Loss > VaR_alpha]; tail loss average.

Bayes Rule: P(A|B)=P(B|A)P(A)/P(B); updates probabilities with evidence.

Entropy: H=-sum(p_i ln p_i); uncertainty/concentration measure.

Mutual Information: MI(X,Y)=sum p(x,y)ln[p(x,y)/(p(x)p(y))]; nonlinear dependence measure.

Sharpe t-Stat Approx: t ≈ Sharpe*sqrt(years); rough significance for independent returns.

Autocorrelation: rho_k = corr(r_t,r_{t-k}); momentum/reversal detection.

AR(1): X_t = c + phi*X_{t-1}+epsilon_t; persistence/mean-reversion model.

Half-Life: h=-ln(2)/ln(phi); decay time for AR(1) mean reversion.

Cointegration Spread: s_t = y_t - beta*x_t; stationary relative-value residual.

Crowding Risk: many participants own same trade; exit door too small.

Correlation Spike Risk: diversification fails during crisis.

Leverage Risk: borrowed exposure magnifies losses and liquidation probability.

Margin Call Risk: broker forced liquidation after equity falls.

Borrow Recall Risk: short position forced closed.

Short Squeeze Risk: short position rapidly loses as price rises.

Volatility Crush Risk: long options lose as IV collapses after event.

Theta Bleed: options lose value with time decay.

Slippage Risk: actual fills worse than assumed.

Lookahead Bias: accidental use of future data.

Survivorship Bias: excluding delisted losers inflates backtest.

Selection Bias: picking universe after knowing winners.

Overfitting: model fits noise, not signal.

Data Mining Bias: too many tested strategies create false positives.

Regime Dependence: strategy only works in certain market states.

Tail Risk: rare extreme losses dominate long-run outcome.

Negative Skew: frequent small wins with rare large losses.

35) MINIMUM VIABLE RESEARCH WORKFLOW
================================================================================

1. Define hypothesis: exact reason edge should exist.

2. Define universe: stocks, liquidity threshold, price threshold, survivorship-free.

3. Define signal: formula using only point-in-time available data.

4. Define target: next-day/week/month return, risk-adjusted return, or trade outcome.

5. Split data: train/validation/test by time, not random.

6. Include costs: spread, slippage, fees, borrow, market impact.

7. Test robustness: parameters, subperiods, regimes, sectors, liquidity buckets.

8. Test significance: t-stat, bootstrap, probabilistic/deflated Sharpe.

9. Control exposure: beta, sector, factor, concentration, correlation.

10. Size positions: volatility/risk budget/Kelly fraction.

11. Simulate execution: delay, partial fills, volume caps.

12. Paper trade live: verify data latency, fills, behavior.

13. Trade small: measure live slippage and alpha decay.

14. Monitor decay: rolling Sharpe, hit rate, drawdown, IC, factor exposure.

15. Kill or resize: stop when live behavior violates expected distribution.
